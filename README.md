# Roast My Startup

Get honest feedback on your startup idea, business plan, and strategy. No fluff. No false hope.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deployment

The project is connected to Vercel. Pushing to the `main` branch automatically creates a production deployment.

## Stack

- React
- Vite
- Vercel
